# Two games written in python
 - pong - The classical ping pong game.
 - spacemission - Avoid rocks when flying your space ship to collect money in space.

## Install
python -m venv venv  
. venv/bin/activate

pip install pygame

## Play
python pong.py  
or  
python spacemission.py
